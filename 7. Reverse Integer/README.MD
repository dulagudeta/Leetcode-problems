<h2><a href="https://leetcode.com/problems/reverse-integer/">7. Reverse Integer</a></h2> <img src="https://img.shields.io/badge/Difficulty-Medium-orange" alt="Difficulty: Medium" /> <hr> <p>Given a signed 32-bit integer <code>x</code>, return <code>x</code> with its digits reversed. If reversing <code>x</code> causes the value to go outside the 32-bit signed integer range <code>[-2<sup>31</sup>, 2<sup>31</sup> - 1]</code>, return <code>0</code>.</p> <p><strong>Note:</strong> The environment does not allow storing 64-bit integers.</p> <h3>Example 1:</h3> <pre> <strong>Input:</strong> x = 123 <strong>Output:</strong> 321 </pre> <h3>Example 2:</h3> <pre> <strong>Input:</strong> x = -123 <strong>Output:</strong> -321 </pre> <h3>Example 3:</h3> <pre> <strong>Input:</strong> x = 120 <strong>Output:</strong> 21 </pre> <h3>Constraints:</h3> <ul> <li><code>-2<sup>31</sup> &lt;= x &lt;= 2<sup>31</sup> - 1</code></li> </ul>
