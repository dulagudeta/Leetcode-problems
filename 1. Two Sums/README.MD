Two Sum</a></h2> <img src="https://img.shields.io/badge/Difficulty-Easy-brightgreen" alt="Difficulty: Easy" /> <hr> <p>You are given an array of integers <code>nums</code> and an integer <code>target</code>.</p> <p>Return the indices of the two numbers such that they add up to <code>target</code>.</p> <p>You may assume that each input has exactly <strong>one valid solution</strong>, and you may not use the same element twice.</p> <p>You can return the answer in any order.</p> <h3>Example 1:</h3> <pre> <strong>Input:</strong> nums = [2,7,11,15], target = 9 <strong>Output:</strong> [0,1]

<strong>Explanation:</strong>
Because nums[0] + nums[1] = 2 + 7 = 9.
